Mapa interactivo de Covid-19

Este es un proyecto con fines didacticos que tiene como fin mostrar como añadir una API de Google Maps y poder añadir datos de manera visual en tiempo real. El mapa actualiza los datos cada 24 hrs. Así que es posible que estos datos no coicidan siempre en tiempo. El proyecto esta diseñado con el principio "Desktop First" por lo que tiene como prioridad verse mejor en pantallas de escritorio que dispotivios moviles. Este proyecto funciona en cualquier navegador.
